Echo – Multilingual Text-to-Speech Web Application
______________________________________________________

Echo is a lightweight web application that converts multilingual text (for example, Hindi-English mixed input) into natural-sounding speech.
Built using Flask, gTTS (Google Text-to-Speech), and styled with Neumorphism, it delivers an elegant and responsive user experience.

_________
Features
_________

Supports multilingual input (e.g., "Hello कैसे हो?")

Generates natural human-like speech

Simple and modern Neumorphic UI

Lightweight Flask backend with minimal dependencies

Fully open-source and deployable on Render

___________
Tech Stack
___________

Frontend: HTML, CSS (Neumorphism), JavaScript
Backend: Flask (Python)
Text-to-Speech Engine: gTTS (Google Text-to-Speech)
Hosting: Render (Free Tier)

_______________________
Setup and Installation
_______________________

## Clone the repository

git clone https://github.com/rosnnn/Echo.git
cd Echo


## Create a virtual environment

python -m venv venv
source venv/bin/activate     # For Linux/Mac
venv\Scripts\activate        # For Windows


## Install dependencies

pip install -r requirements.txt


## Run the Flask app

python app.py


## Open in your browser

http://127.0.0.1:5000
